# La Esquina Gastrobar Website

Sitio web estático de La Esquina Gastrobar (Constanza, RD), preparado para producción.

## Estado
- Producción lista.
- Repositorio publicado en GitHub.
- Release inicial etiquetado como `v1.0.0`.

## Estructura principal
- `index.html` — Inicio
- `servicios.html` — Servicios
- `menus.html` — Menú y sección destacada
- `galeria.html` — Galería
- `nosotros.html` — Nosotros
- `contacto.html` — Contacto
- `404.html` — Página de error personalizada

## Archivos de producción
- `robots.txt`
- `sitemap.xml`
- `manifest.webmanifest`
- `.htaccess`
- `DEPLOY.md`
- `CHANGELOG.md`

## Deploy
Consulta pasos detallados en `DEPLOY.md`.

### Resumen rápido
1. Subir todos los archivos al hosting.
2. Verificar dominio final `https://www.laesquinard.com`.
3. Probar rutas principales y la página 404.
4. Limpiar caché del hosting/CDN.

## Notas
- `client-original.html` se conserva solo como archivo histórico de extracción, no como página pública operativa.
