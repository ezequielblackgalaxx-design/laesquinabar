# Changelog

Todos los cambios relevantes de este proyecto se documentan en este archivo.

## [v1.0.0] - 2026-03-18

### Added
- Sitio estático completo de La Esquina Gastrobar con páginas:
  - `index.html`
  - `servicios.html`
  - `menus.html`
  - `galeria.html`
  - `nosotros.html`
  - `contacto.html`
- Página `404.html` personalizada.
- Archivos de preparación para producción:
  - `robots.txt`
  - `sitemap.xml`
  - `manifest.webmanifest`
  - `.htaccess`
  - `DEPLOY.md`
- Sección destacada de cerveza artesanal en `menus.html` con animaciones e interacción de puntero.
- Efectos visuales equivalentes en la sección “Nuestros servicios principales” de `servicios.html`.

### Changed
- Navegación y enlaces internos unificados entre páginas principales.
- Ajustes de metadatos en páginas para favicon, `theme-color` y manifest.

### Notes
- El archivo `client-original.html` se mantiene como fuente histórica de extracción y no forma parte de la página pública operativa.
